# Resources

## Books

- **Crafting Interpreters** by Robert Nystrom
  - Excellent, free online book: [craftinginterpreters.com](https://craftinginterpreters.com)
  - Covers tree-walking interpreters and bytecode VMs

- **Writing an Interpreter in Go** by Thorsten Ball
  - Practical, hands-on approach

- **Engineering a Compiler** by Cooper & Torczon
  - Comprehensive academic textbook

## Rust Resources

- **The Rust Programming Language** ("The Book")
  - [doc.rust-lang.org/book](https://doc.rust-lang.org/book)

- **Rust by Example**
  - [doc.rust-lang.org/rust-by-example](https://doc.rust-lang.org/rust-by-example)

## Tools Used

- **pest** - PEG parser generator for Rust
  - [pest.rs](https://pest.rs)
  - [docs.rs/pest](https://docs.rs/pest)

- **inkwell** - Safe LLVM wrapper for Rust
  - [github.com/TheDan64/inkwell](https://github.com/TheDan64/inkwell)

- **LLVM** - Compiler infrastructure
  - [llvm.org](https://llvm.org)
  - [llvm.org/docs/LangRef.html](https://llvm.org/docs/LangRef.html) - LLVM IR reference

## Academic Papers

- **A Nanopass Framework for Compiler Education** - Sarkar, Waddell, Dybvig
- **Linear Scan Register Allocation** - Poletto, Sarkar
- **Simple and Efficient Construction of SSA Form** - Braun et al.
